# image-resources
# 🎨 p5.js 图形资源库

这个仓库收集了我为 p5.js 开源项目创作的一系列图形资源，包括图标、纹理、背景图等，所有资源均以**非商业使用**为前提免费开放。
本仓库中的所有图形资源采用以下许可：

⚠️ CC BY-NC 4.0 (知识共享-署名-非商业性使用 4.0 国际许可协议)
这意味着你可以：
✅ 自由使用：在个人项目中复制、修改、分发这些资源
✅ 开源项目使用：可用于个人或开源的 p5.js 项目

但你必须遵守：
❌ 不得商用：不得将这些资源用于任何商业目的（包括但不限于出售、商业产品、付费应用等）
📝 署名致谢：使用时须注明来源（提供本仓库链接 或 标注@流白Livo）
🔗 相同方式共享：修改后的资源也需使用相同的许可协议发布

📖 查看完整的许可协议：[CC BY-NC 4.0](https://creativecommons.org/licenses/by-nc/4.0/deed.en)

## 🚀 使用方式
### 通过 jsDelivr CDN 引用
所有资源均可通过以下格式的 CDN 链接直接引用：
https://cdn.jsdelivr.net/gh/xxoogreymon-prog/image-resources@main/文件名/图片名
